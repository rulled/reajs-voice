# reajs chain for voice

этот jsfx плагин представляет собой цепочку эффектов для обработки голоса в реальном времени.

## установка

### для reaper

1.  скопируйте весь код из файла.
2.  внутри reaper откройте меню `actions` > `show action list...`.
3.  в поиске найдите `jsfx: create new jsfx`.
4.  выберите действие и нажмите `run`.
5.  дайте файлу имя (например, `voice chain`) и сохраните его.
6.  откроется текстовый редактор с кодом-шаблоном. удалите все и вставьте скопированный код.
7.  сохраните файл (ctrl+s или cmd+s).

плагин появится в вашем списке fx.

### для других программ (obs, fl studio и т.д.)

1.  скачайте и установите набор плагинов `reaplugs` с официального сайта:
    [https://www.reaper.fm/reaplugs/](https://www.reaper.fm/reaplugs/)

2.  переместите файл с этим плагином (например, `voice chain`, важно, что у файлов нет расширения) в папку с js эффектами reaplugs. путь обычно такой:
    `c:\program files\vstplugins\reaplugs\js\effects\`

3.  внутри папки `effects` можно создать свою папку (например, `voice`) для удобства и поместить файл туда.

4.  в вашей программе (например, obs) добавьте vst плагин `reajs`.

5.  в окне плагина `reajs` нажмите `load...` и выберите ваш файл эффекта.

## использование

плагин можно найти в окне добавления эффектов (fx) в категории js (для reaper) или как vst плагин reajs (для других программ).

### рекомендуемая цепочка

для наилучшего результата используйте шумоподавление перед этим плагином. например, `rnnoise`.

правильная последовательность:
`rnnoise` > `reajs (этот плагин)`

скачать rnnoise можно здесь:
[https://github.com/werman/noise-suppression-for-voice](https://github.com/werman/noise-suppression-for-voice)

## настройки

все модули можно включать и отключать.

*   **gate**: убирает тихие звуки (шум, дыхание).
    *   `gate threshold`: порог срабатывания. все, что тише этого значения, будет вырезано.
    *   `gate attack`: как быстро гейт открывается.
    *   `gate release`: как быстро гейт закрывается.

*   **de-esser**: подавляет шипящие звуки ("с", "ц").
    *   `de-esser freq`: частота, на которой происходит подавление.
    *   `de-esser reduction`: насколько сильно подавлять шипящие.

*   **eq**: эквалайзер для коррекции частот.
    *   `hpf cutoff`: обрезает самый низ, убирая гул.
    *   `low mud cut (250hz)`: убирает мутность в голосе.
    *   `presence boost (3khz)`: добавляет разборчивости и "присутствия".
    *   `air boost (10khz)`: добавляет "воздуха" и яркости.

*   **optical comp**: мягкий компрессор, выравнивает общую динамику.
    *   `peak reduction`: степень сжатия.
    *   `output gain`: компенсация громкости после сжатия.

*   **fet comp**: быстрый компрессор, ловит резкие пики.
    *   `fet threshold`: порог срабатывания компрессора.
    *   `fet ratio`: соотношение сжатия.
    *   `fet attack`: скорость срабатывания.
    *   `fet release`: скорость восстановления.

*   **saturation**: добавляет теплоту и гармоники, делая звук плотнее.
    *   `tube drive`: степень насыщения.
    *   `sat mix %`: процент подмешивания эффекта к чистому сигналу.

*   **exciter**: добавляет яркости и "искры" высоким частотам.
    *   `excite amount %`: количество эффекта.

*   **limiter**: ограничивает сигнал, не давая ему превысить заданный уровень.
    *   `limiter ceiling`: максимальный уровень громкости, выше которого сигнал не пройдет.
    *   `output level %`: итоговая громкость плагина.
